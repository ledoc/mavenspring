Basics:

Bootstrap for creating other projects using Spring, Hibernate, Jetty server and Maven.

Liscense:

It can be used, copied and distributed by any parties.

How to use?
1. Install Maven if you haven't 
2. Clone this project
3. Import in Eclipse
4. Configure app.properties file with MySql database connection
4. Search JettyServerStartup.java and run main method providing necessary arguments
5. Then you are ready adding other components.
